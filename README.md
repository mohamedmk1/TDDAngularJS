
## Project TDDAngularJS

**FOR** Craft developers

**WHO WISH** To develop functions using TDD approach

**OUR PRODUCT IS** A service voucher containing customer, products and delivery sections

## Main features to implement

- Implement service voucher validation using TDD approach

## Technical Stack:

- AngularJS: Framework de développement JavaScript
- Karma: Terminal JavaScript permettant d'éxécuter les tests
- Jasmine Framework: Framework de test JavaScript

## Developing

- Install packages: npm install
- Build: npm run build
- Build files and watch: npm start

## Testing

- Single run: npm test
- Live mode (TDD style): npm run test-watch
